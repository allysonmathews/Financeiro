# Financeiro

Aplicativo mobile de gestão financeira (Atlas, Franquias, Pessoal) com interface de chat com IA.

## Stack

- **Expo** (React Native)
- **NativeWind** (Tailwind CSS)
- **Lucide React Native** (ícones)
- **Expo Router** (navegação por arquivos)

## Como rodar

```bash
npm install
npx expo start
```

Depois escaneie o QR code com o app **Expo Go** (Android/iOS) ou pressione `a` (Android) / `i` (iOS) no terminal.

## Estrutura

- `app/index.tsx` — Tela inicial com 3 cards (Atlas, Franquias, Pessoal) e barra de chat com IA
- `app/_layout.tsx` — Layout raiz (Stack, StatusBar)

Design: gradiente azul escuro, cards brancos com cantos arredondados e sombra suave, alinhado ao visual Atlas Fibra.
